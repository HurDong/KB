기프티콘 관리 + 거래 플랫폼 :

조사 :
카카오 디벨로퍼에서 확인 결과 선물하기 관련 api는 없는 것으로 파악.  카카오톡 선물하기 for biz가 있는 것 같은데, 해당 api로는 기프티콘 생성 및 구입이 가능할뿐 기프티콘 교환과는 힘들어보일 것으로 보임.

CRUD
- Create : 기프티콘 거래 게시물 생성
- Read : 자신의 기프티콘 조회, 기프티콘 거래 게시물 조회
- Update : 기프티콘 거래시 교환, 기프티콘 거래 게시물 수정
- Delete : 기프티콘 사용시 삭제, 기프티콘 거래 게시물 삭제

자신의 기프티콘 조회 :
- 디폴트 정렬은 유효기간 오름차순
- 정렬 이름순 오름내림, 유효기간 오름내림 가능
- 유호기간이 임박한 기프티콘에 관해서 시각적으로 표시
- 기프티콘 상세 조회 가능

기프티콘 상세 조회 :
- 해당 기프티콘을 바로 거래 게시물에 올리는 버튼 추가
- 기프티콘 상세한 정보 (유효기간, 이름, 교환한 사람(없다면 원래 준 사람),...)

기프티콘 거래 게시물 :
- 게시물 번호, 아이디, 교환할 것 or 교환하고 싶은것, 내용
- 교환하기 버튼 생성 -> 바로 해당 유저와 1대1 채팅으로 교환하는 화면으로 or 소통없이 거래 원하는 기프티콘만 올리고 교환

========================================

설문 조사 문항 :
- 기프티콘을 자주 보내거나 받는지
- 버리는 기프티콘이 있는지 -> 환불을 하는지
- 쓸 일 없는 기프티콘을 그에 상응한 가치의 다른 기프티콘으로 교환할 수 있으면 할 것인지
