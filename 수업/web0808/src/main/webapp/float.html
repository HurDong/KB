첫째 줄에 queuestack을 구성하는 자료구조의 개수 $N$이 주어진다. ( $1 \leq N \leq 100\,000$) 둘째 줄에 길이 $N$의 수열 $A$가 주어진다. $i$번 자료구조가 큐라면 $A_i = 0$, 스택이라면 $A_i = 1$이다. 셋째 줄에 길이 $N$의 수열 $B$가 주어진다. $B_i$는 $i$번 자료구조에 들어 있는 원소이다. ( $1 \leq B_i \leq 1\,000\,000\,000$) 넷째 줄에 삽입할 수열의 길이 $M$이 주어진다. ( $1 \leq M \leq 100\,000$) 다섯째 줄에 queuestack에 삽입할 원소를 담고 있는 길이 $M$의 수열 $C$가 주어진다. ( $1 \leq C_i \leq 1\,000\,000\,000$) 입력으로 주어지는 모든 수는 정수이다.
